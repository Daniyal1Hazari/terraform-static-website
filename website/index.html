🚀 Terraform Static Website Deployment

A complete Infrastructure as Code (IaC) project demonstrating automated provisioning, deployment, and hosting of a static website on AWS using modern DevOps practices.

☁️ AWS Cloud Infrastructure

Infrastructure provisioned using Terraform, including an Amazon EC2 instance, IAM Role, Security Group, and automated server configuration.

⚙️ Infrastructure as Code

All cloud resources are managed through Terraform, ensuring consistent, repeatable, and version-controlled infrastructure deployment.

🔄 CI/CD Automation

GitHub Actions automates validation and deployment, enabling reliable and efficient infrastructure provisioning from a centralized Git repository.

🛡️ Secure Cloud Architecture

AWS IAM Roles and Security Groups are configured to follow cloud security best practices while providing controlled public HTTP access.

🛠 Technology Stack

AWS EC2 | Terraform | GitHub Actions | Linux | Nginx | HTML | CSS | JavaScript
